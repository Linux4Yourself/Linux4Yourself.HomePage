Linux для себя

В своём большинстве, книга основана на наработках замечательного проекта Linux From Scratch. Однако, позиционируется как её ответвление.

Отличительными особенностями является более широкое раскрытие некоторых важных аспектов и предоставление наиболее исчерпывающией информации по различным приемам создания операционной системы из исходного кода.

Начните собственное путешествие в мир Linux прямо сейчас!

Основная часть

Основная часть посвящена созданию минимальной рабочей базовой системы Linux созданной из исходного кода.
